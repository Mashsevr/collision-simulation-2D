

1. Краткое описание:
Модель на Python визуализирует столкновения тел в двумерном пространстве. Поддерживаются упругие и неупругие взаимодействия, отображаются векторы, траектории, энергия, импульс и графики изменения величин во времени.

2. Файлы:
- collision_simulation.py основной код симуляции
- readme.txt текущая инструкция

3. Зависимости:
Для запуска требуетсЯ Python 3.10+ и библиотеки:
- pygame
- matplotlib

Установка:
pip install pygame matplotlib

4. Как запустить:
1) Открой терминал в папке проекта
2) Выполни команду:
   python collision_simulation.py

5. Управление в программе:
- E переключение между упругим/неупругим столкновением
- V включить/выключить векторы
- T следы движения
- SPACE пауза
- G построить графики энергии и импульса
